# Security Policy

## Supported Versions

Only the latest version of Snekbooru is officially supported with security updates. We recommend all users stay on the latest release to ensure they have the most recent security patches and stability improvements.

| Version   | Supported          |
| --------- | ------------------ |
| >= 6.0.0  | :white_check_mark: |
| < 6.0.0   | :x:                |

## Reporting a Vulnerability

We take the security of Snekbooru seriously. If you find a security vulnerability, please do not disclose it publicly. Instead, please report it through one of the following channels:

1.  **Discord**: Join our [Official Discord Server](https://discord.gg/BqNxn7ftqn) and contact any of the devs via Direct Message.
2.  **GitHub Issues**: If the vulnerability is not critical (e.g., a minor bug that doesn't expose sensitive data), you can open a GitHub Issue. For critical vulnerabilities, please use the private channels mentioned above.

When reporting a vulnerability, please provide:
*   A detailed description of the issue.
*   Steps to reproduce the vulnerability.
*   The potential impact of the vulnerability.
*   The version(s) of Snekbooru affected.

We will acknowledge your report as soon as possible and work towards a resolution.
